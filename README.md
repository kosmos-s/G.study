## Library Structure

```text
G.study
├─ portfolio/       완성된 프로젝트 포트폴리오
│  └─ 프로젝트명/
│     ├─ README.md              프로젝트 메인 페이지
│     ├─ technical-details.md   기술 구성과 구현 내용
│     ├─ troubleshooting.md     문제 발생 및 해결 과정
│     ├─ evidence.md            화면 자료와 측정 결과
│     └─ images/                공개용 이미지
│
├─ projects/        개발 중인 프로젝트
├─ study/           학습 기록
├─ experiments/     기술 실험과 검증
└─ assignments/     수업 실습과 과제
```

## How to Browse

1. 위의 **Portfolio Library**에서 프로젝트 카드를 선택합니다.
2. 프로젝트 메인 페이지에서 핵심 성과와 전체 흐름을 확인합니다.
3. 더 자세한 내용은 기술 구성, 문제 해결, 증거 자료 문서로 이동해 확인합니다.

새로운 프로젝트가 포트폴리오 수준으로 정리될 때마다 이 라이브러리에 카드 형태로 추가합니다.
