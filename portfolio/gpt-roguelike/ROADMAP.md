- 사망·승리 결과 처리
- Soul 보상과 영구 강화 구조

### RPG와 콘텐츠

- 전사·궁수·마법사·암살자·수호자 직업
- 직업별 Q·E·R
- 무기와 패시브 아이템
- 아이템 빌드 UI
- 인벤토리, 장비 장착, 판매
- F~S 직업 무기 가챠와 확률표
- 원거리 적 투사체
- 랭킹
- 플레이어 거래 프로토타입
