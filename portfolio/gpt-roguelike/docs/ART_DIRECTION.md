# Art Direction

## 장르

3D Vampire Survivors 스타일 생존 전투와 Hades 스타일 로비 성장을 결합한 다크 판타지 로그라이트.

## 환경

- 로비: 어두운 마법 성소, 포탈, 제단, 무기 전시대
- 생존맵: 황량한 숲, 폐허, 묘지, 마법 안개

## UI 컬러

- 기본 패널: 검정 / 남색
- Soul: 청록
- Gold: 금색
- EXP: 보라
- HP: 붉은색

## 이펙트

- Sword: 파란 참격
- Spear: 청록 관통선
- Dagger: 노란 빠른 플래시
- Staff: 보라 마법진과 폭발
- Axe: 주황 충격파

향후 UI와 맵은 코드 생성보다 Roblox Studio에서 직접 배치하고 조정할 계획입니다.
