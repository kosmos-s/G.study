# GPT Roguelike

> Roblox Studio와 AI를 활용해 제작 중인 **3D 뱀파이어 서바이벌 전투 + 하데스식 로비 성장 로그라이트** 프로젝트입니다.

## 게임 컨셉

- **장르:** 3D 생존 액션 로그라이트 · RPG
- **플랫폼:** Roblox
- **개발 형태:** 개인 프로젝트 · 바이브 코딩
- **아트 방향:** 다크 판타지 생존 로그라이트
- **로비:** 어두운 마법 성소, 제단, 포탈, 무기 전시대
- **생존맵:** 황량한 숲, 폐허, 묘지, 마법 안개
- **성장 구조:** 생존 전투에서 얻은 보상으로 로비에서 영구 성장

```text
로비에서 직업·장비 선택
→ 포탈을 통해 생존 시작
→ 자동 공격과 직업별 Q/E/R 사용
→ EXP·Gold 획득 및 레벨업
→ 무기·아이템 빌드와 진화
→ 사망 또는 승리
→ Soul 보상과 영구 성장
```

## 프로젝트를 시작한 이유

사람들이 바이브 코딩으로 웹사이트, 앱, 게임 등 다양한 결과물을 만드는 모습을 보면서 저도 직접 무언가를 만들어 보고 싶었습니다.

그중 게임 제작에 도전하기로 했고, Unity처럼 새로운 엔진을 처음부터 배우기보다 예전에 YouTube 영상을 따라 게임을 만들어 본 경험이 있는 Roblox Studio를 선택했습니다. 익숙한 환경에서 AI를 활용해 실제로 플레이할 수 있는 게임 하나를 완성해 보자는 목표로 시작했습니다.

## 주요 기능

- 로비와 생존맵
- 포탈 생존 시작
- 자동 공격
- 직업 5종과 직업별 Q/E/R 스킬
- 직업 변경 비용과 직업별 무기
- F~S 무기 가챠, 확률표, 천장·조각 구조
- Robux 가챠 준비 UI (`ProductId = 0`, 실제 결제 미연동)
- EXP·Gold 드롭과 자동 흡수
- 인게임 아이템 빌드 UI
- 무기 진화 시스템
- 인벤토리, 장비 장착, 판매
- 랭킹과 플레이어 거래 프로토타입
- 원거리 몬스터 투사체
- Soul 보상과 영구 강화
- 사망·승리 결과창
- 숫자키 1·2·3 레벨업 선택
- 공격 사거리 표시

> 위 기능 중 일부는 구현 후 회귀 테스트에서 실패하거나 재검증 대기 상태입니다. 실제 통과·실패 결과는 [`docs/TESTING.md`](docs/TESTING.md)에 구분해 기록합니다.

## 기술 스택

| 기술 | 사용 목적 | 상태 |
|---|---|---|
| Roblox Studio | 맵, 오브젝트, UI와 플레이 테스트 | 현재 사용 |
| Luau | 클라이언트·서버 게임 로직 | 현재 사용 |
| DataStore | 직업, 장비, Soul 등 영구 데이터 | 구현 및 안정화 중 |
| RemoteEvent | 클라이언트와 서버 간 요청·상태 동기화 | 현재 사용 |
| ModuleScript | 직업, 무기, 아이템, 적 데이터 분리 | 현재 사용 |
| Rojo | VS Code 기반 코드 구조와 버전 관리 | 도입 예정 |
