# Screenshots

GPT Roguelike의 실제 Roblox Studio 및 플레이 화면을 보관하는 폴더입니다.

## 추가할 자료

- 로비 전체 화면
- 포탈 상호작용
- 생존 전투
- 자동 공격과 직업별 스킬
- 레벨업 선택 UI
- 인벤토리·가챠·랭킹·거래 UI
- 사망·승리 결과 화면

## 파일 이름 규칙

```text
v버전-기능-설명.png
```

예시:

```text
v2.7.2-lobby-quick-menu.png
v2.7.2-survival-auto-attack.png
```

이미지를 올리기 전에 Roblox 사용자 이름, 계정 정보, 비공개 에셋 ID 등 공개하면 안 되는 정보가 포함되지 않았는지 확인합니다.
