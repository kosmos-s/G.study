
## 촬영 기준

- UI가 잘리지 않도록 전체 화면 크기 확인
- 동일 기능은 가장 이해하기 쉬운 화면만 선택
- 테스트용 디버그 UI와 불필요한 Output 창 숨기기
- 기능 설명과 실제 화면이 일치하는지 확인
- 전후 비교는 같은 위치와 화면 비율로 촬영

## 공개 전 확인

- Roblox 사용자 이름
- 채팅과 알림
- 계정 정보
- 비공개 에셋 ID
- Product ID
- 로컬 파일 경로
- 개인 서버 또는 내부 정보

민감정보가 보이면 가린 뒤 실제 이미지 파일을 다시 확인합니다.
