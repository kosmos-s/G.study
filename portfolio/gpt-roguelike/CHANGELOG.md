# 변경 기록

이 문서는 GPT Roguelike의 패치별 변경 사항을 기록합니다.

완료된 구현, 수정 시도, 테스트 대기 상태를 구분하며 실제로 확인하지 않은 기능은 완료로 표시하지 않습니다.

## Unreleased

### 진행 중

- v2.7.2 상태 동기화 수정 사항 재검증
- 직업·무기 즉시 갱신 확인
- 재접속 후 직업과 장착 무기 복원 확인
- 이동속도와 사거리 아이템의 실제 스탯 적용 확인
- 무기 진화 조건과 진화 카드 등장 확인
- UI·맵·전투 이펙트 Visual Rebuild 준비

## v2.7.2 — Core State Fix

### 변경

- 직업 변경 후 HUD와 장착 무기를 즉시 갱신하도록 수정 시도
- 재접속 시 직업과 장착 무기 복원 로직 수정 시도
- 이동속도와 사거리 아이템 적용 로직 수정 시도
- 진화 조건 판정과 카드 등장 흐름 수정 시도

### 상태

- 코드 반영 후 Roblox Studio 회귀 테스트 필요
- 성공 여부는 `docs/TESTING.md`에서 관리

## v2.7.1

### 변경

- 최초 직업 선택 흐름 보완
- 직업 변경 비용 처리
- 레벨업 선택에 숫자키 1·2·3 입력 추가
- 인벤토리 열기 문제 수정
- 사거리 표시 링 위치 조정
- ClientController 문자열 구문 오류 수정

## v2.7.0 — Mega Systems

### 추가

- 원거리 몬스터 투사체
- 인벤토리, 장착, 판매
- 랭킹
- 플레이어 거래 프로토타입
- 가챠 확률표
- Robux 가챠 준비 UI

### 수정

- ProjectileManager와 RemoteEvents 누락으로 발생한 Infinite Yield 문제 대응

## v2.6.x — RPG Economy

### 추가

- 직업 시스템과 직업별 Q·E·R
- RPG 스탯과 하단 HUD
- 인게임 아이템과 진화 조합
- 로비 상점
- F~S 직업 무기 가챠
- 천장과 중복 조각 구조

## v2.5.x — Content & Visual Expansion

### 추가

- 무기 레벨과 진화 기반
- 적 종류와 보스 패턴
- 패시브 아이템과 빌드
- 퀘스트·업적 기반
- Visual Rebuild 실험

## v2.0.0 — Survival Roguelite

### 추가

- 로비와 생존 전투 맵
- 포탈 상호작용
- 시간 기반 몬스터 스폰과 추적
- EXP 보석과 Gold 코인
- 자동 흡수와 획득 피드백
- Soul 결과 보상과 영구 강화
- DataStore 저장 구조
