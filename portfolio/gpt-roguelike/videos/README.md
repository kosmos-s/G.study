# Videos

GPT Roguelike의 플레이 영상과 기능 시연 자료를 관리하는 폴더입니다.

GitHub 저장소에는 용량이 작은 미리보기 파일이나 외부 영상 링크 안내만 보관하고, 큰 원본 영상은 별도 저장소 또는 영상 플랫폼을 사용합니다.

## 권장 영상

- 30초 핵심 게임 루프
- 1~2분 전체 플레이 데모
- 패치 전후 비교
- 버그 재현 및 수정 결과

## 파일 이름 규칙

```text
v버전-기능-설명.mp4
```

영상 공개 전 사용자 이름, 채팅, 알림, 개인 파일 경로 등 개인정보가 포함되지 않았는지 확인합니다.
