## 권장 영상 구성

### 30초 핵심 영상

```text
로비 준비
→ 포탈 진입
→ 자동 공격과 스킬
→ 레벨업 선택
→ 결과창과 Soul 성장
```

### 1~2분 플레이 데모

1. 직업과 장비 선택
2. 생존맵 진입
3. 자동 공격과 직업 스킬
4. EXP·Gold 획득
5. 아이템 빌드와 진화 시도
6. 인벤토리 또는 가챠 UI
7. 사망·승리 결과
8. 로비 영구 성장

## 파일 이름 규칙

```text
v버전-기능-설명.mp4
```

예시 이름은 실제 영상 파일이나 링크를 추가할 때 사용합니다.

```text
v2.7.3-core-game-loop.mp4
v2.7.3-foundation-stability-test.mp4
```

## 저장 방식

- GitHub에는 용량이 작은 미리보기 또는 외부 영상 링크 안내만 보관
- 큰 원본 영상은 별도 로컬 저장소나 영상 플랫폼에서 관리
- README에는 실제 공개가 확인된 링크만 추가

## 공개 전 확인

- Roblox 사용자 이름과 채팅
- Windows 알림과 개인 파일 경로
- 계정·이메일 정보
- 비공개 서버 또는 에셋 정보
- 녹화 중 노출된 API 키, Product ID, 내부 데이터

민감정보를 제거하고 영상이 정상 재생되는지 확인한 뒤 링크합니다.
